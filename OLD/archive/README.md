# Archive Directory

This directory contains old code that has been refactored or is no longer actively used.

## Contents

- `old_Analysis_capitalized/`: Old Analysis module with hardcoded data paths
- `old_scripts/`: Standalone scripts that may still have useful logic but need updating
- `old_mutual_information/`: Various versions of MI implementation before consolidation

## Current Active Code

The actively maintained code is in:
- `src/core/`: Core quantum computing functionality (Magic, Entanglement, Entropy, RandomUnitaries)
- `src/analysis/`: Statistical analysis tools (mutual information)
- `src/utils/`: Utility functions (numerical integration)
- `src/circuits/`: Circuit generation (placeholder for future development)

DO NOT use code from this archive directory without careful review and testing.
